## PROGETTO PROGRAMMAZIONE DI SISTEMA A.A. 2018/19

Esercitazione 3 contiene un'implementazione semplificata dell'algoritmo **CRDT** con strategia di allocazione adattiva (**LSEQ**).
L'esercizio simula l'azione di due editor distinti che comunicano tramite un server.
L'algoritmo è implementato nella classe **SharedEditor**.
